# Bot Discord (Node.js)

## Prérequis
- Node.js 18+ recommandé

## Configuration
1. Copie le fichier `.env.example` vers `.env`
2. Remplis les variables :
   - `DISCORD_TOKEN`
   - `DISCORD_CLIENT_ID`
   - `RIOT_API_KEY` (optionnel pour le moment)

## Installation
```bash
npm install
```

## Enregistrer les commandes slash
```bash
npm run register
```

## Lancer le bot
```bash
npm run start
```

## Commandes
- `/ping` : répond "Pong!"
